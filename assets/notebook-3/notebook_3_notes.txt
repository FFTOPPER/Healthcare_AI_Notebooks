BERT-Based Customer Review Sentiment Analysis - Notes
====================================================

Project Objective
-----------------
This notebook demonstrates how to fine-tune a BERT-family model
(DistilBERT) for binary sentiment classification of customer reviews.

The model predicts whether a review expresses:
- Positive sentiment (1)
- Negative sentiment (0)

Dataset Details
---------------
Dataset Type    : Synthetic Customer Review Dataset
Total Samples   : 500
Features        :
    - review
    - label

Class Distribution
------------------
- Negative Reviews (0): 267
- Positive Reviews (1): 233

Model Details
-------------
Base Model      : distilbert-base-uncased
Architecture    : DistilBERT
Task            : Sequence Classification
Number of Labels: 2
Framework       : Hugging Face Transformers

Training Configuration
----------------------
Epochs                : 2
Train/Test Split      : 80/20
Training Samples      : 400
Testing Samples       : 100
Batch Size            : 16
Learning Rate         : 2e-5
Weight Decay          : 0.01
Evaluation Strategy   : Every Epoch
Metric for Best Model : F1 Score
